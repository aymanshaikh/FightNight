Recreated a lighter version of Super Smash Brawl called Fight Night
------------------------------------------------------------------------

PROJECT TITLE: Super Smash Brawl Remade: Fight Night
PURPOSE OF PROJECT: Recreate a game, program in Java and apply object oriented principles
VERSION or DATE: January 8th 2017
HOW TO START THIS PROJECT: Install Greenfoot and run project file from directory
AUTHORS:Ayman Shaikh
USER INSTRUCTIONS: Press the spacebar to start the game
		   Kirby: WASD to move and spacebar to shoot
		   Metaknight: Arrow keys to move and enter to shoot
		   Player to loose 100% of health grants victory to the other player
		   Random health power ups are dropped so make sure to collect!
